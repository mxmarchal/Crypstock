# Crypstock

## Description
Crypstock is a school project where you can buy/sell and monitor cryptocurrencies.

This project was built usign :
* Xcode 12.2
* Swift 5.3
* SwiftUI

Designed and coded by @mxmarchal

Available in 3 languages:
* English
* French
* Spanish

## Features
* User Management
    * Sign In
    * Sign Up
* Portfolio Management (Crypto)
    * Buy
    * Sell

## Dependancy

* [SQLite.swift](https://github.com/stephencelis/SQLite.swift) - SQLite wrapper for Swift.

## Run

You have all the files to fully run the project with Xcode.

Because there is no "real" back-end, all the users, cryptos and portfolio features are stored in SQLite database in documents folder of your device. (App's Documents)

For now, you only have two cryptocurrencies availables :
* Bitcoin
* Ethereum
